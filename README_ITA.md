# Unity AR Foundation Example
Progetto universitario di Unity che implementa una semplice applicazione Android in realtà aumentata utilizzando AR Foundation.

## Come installare l'applicazione
Scaricare il file `.apk` di una release (o pre-release) su un dispositivo mobile Android e installarlo.
I file `.apk` possono essere trovati sia nella cartella `Builds` del progetto che sul relativo repository di GitHub: [AndreaTassi98/Unity-AR-Foundation-Example](https://github.com/AndreaTassi98/Unity-AR-Foundation-Example).

## Come eseguire l'applicazione
Eseguire l'applicazione installata (`Unity AR Foundation Example`).

## Come utilizzare l'applicazione
- Leggere le informazioni nella finestra di benvenuto e premere il pulsante `Dismiss` per chiuderla.
- Selezionare un oggetto da posizionare premendo il pulsante `Select Object` (è possibile cambiarlo in qualsiasi momento).
- Utilizzare la videocamera del dispositivo per individuare un piano nel mondo reale: quando viene individuato un piano e appare l'indicatore di posizionamento, è possibile posizionare l'oggetto scelto in quella posizione toccando in qualsiasi punto dello schermo (è possibile farlo più volte).
- È possibile nascondere i piani individuati premendo il pulsante `Hide Planes` (oppure, se non sono visibili, è possibile visualizzarli premendo il pulsante `Show Planes`).
- È possibile muovere un oggetto posizionato toccandolo, tenendo premuto e spostandolo sui piani individuati.
- È possibile ridimensionare un oggetto posizionato toccando con due dita e allontanandole (per ingrandire l'oggetto) o avvicinandole (per rimpicciolirlo).
- È possibile ruotare un oggetto posizionato toccando con due dita e ruotandole nella direzione in cui si vuole ruotare l'oggetto (oraria o anti-oraria).
- È possibile rimuovere un oggetto posizionato premendo il pulsante `Remove Object` e, successivamente, toccando l'oggetto posizionato che si vuole rimuovere. Una volta terminata la rimozione degli oggetti, premere il pulsante `Stop Removing` per poterne posizonare altri. È possibile anche rimuovere tutti gli oggetti posizionati in una volta sola, premendo il pulsante `Remove All`.
- È possibile chiudere qualsiasi finestra anche premendo il tasto `Indietro` del dispositivo.
- È possibile uscire dall'applicazione premendo il pulsante `Quit` (oppure premendo il tasto `Indietro` del dispositivo quando nessuna finestra è aperta).
